Usage
=====

With bower:

    bower install atimeline

Manually
--------

Download atimeline.js

    curl -O https://raw.github.com/kimmobrunfeldt/atimeline.js/master/src/js/atimeline.js

Add it to HTML:

    <script src="js/atimeline.js"></script>

Development
===========

Clone the repository and run:

    npm install
